# CXCL12-placenta
Repository to reproduce analyses from the preprint "3D imaging of the pregnant uterus reveals an extensively invasive mouse placenta and early CXCL12-CXCR4 requirement"

<h2>Running the Code</h2>
This code requires Python packages: seaborn, numpy, pandas, and matplotlib. 

The code expects to run from the source data file ("Source_Data_CXCL12_placenta.xlsx") in the same directory as the notebooks, where it will pull the respective source data from corresponding sheets in the source data to produce the graphs.
